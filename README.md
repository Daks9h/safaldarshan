[![Live Demo](https://img.shields.io/badge/Live-Demo-brightgreen?style=for-the-badge&logo=vercel)](https://safaldarshan.vercel.app)

> **"Helping everyone on their holy journeys."**

**सफलDarshan** is a digital transformation for the spiritual world. It bridges the gap between devotees and temples across India, replacing long, unpredictable physical queues with a streamlined, peaceful digital experience.

---

## 🌟 Why सफलDarshan?
In India, the spiritual journey is deeply personal. But the logistics—long queues, heat, and uncertainty—often take the focus away from the *Darshan* itself. 
We built this platform to ensure that your energy is spent on **Bhakti**, not on the **Bheed**.

---

## ✨ Features

### 👤 For Devotees
- **Queue-Free Devotion:** Browse temples and book your darshan slots from home.
- **Pooja Pre-Booking:** Detailed info on available rituals and easy booking.
- **Smart Filters:** Search by Temple Name, City, or State.
- **Manage Bookings:** Track your upcoming and past holy visits.

### 🏛️ For Temple Organizers
- **Digital Management:** List your temple and manage slots effectively.
- **Crowd Control:** Real-time visibility of bookings to handle crowd flow.
- **Record Keeping:** Digital records of all poojas and devotees.

## 🚀 Getting Started

### Prerequisites
- Node.js & npm installed
- MongoDB Atlas account (or local MongoDB)

### Setup
1. **Clone the repo:**
   ```bash
   git clone https://github.com/yourusername/safaldarshan.git
   cd safaldarshan
   ```

2. **Install Dependencies:**
   ```bash
   npm run install-all
   ```

3. **Environment Variables:**
   Create a `.env` file in both the `/client` and `/server` folders (refer to `.env.example` in those folders).

4. **Run Development Server:**
   ```bash
   npm run dev
   ```
